# Market Analysis

Analysis of NZ electricity market clearing prices and patterns.

## Contents

- **Complete_Analysis.md/pdf** - Comprehensive analysis of clearing price patterns and optimization opportunities
- **clearing_price_analysis.py** - Analysis script for clearing price data
- **outputs/** - Analysis results and summaries

## Running Analysis

```bash
cd analysis
uv run python clearing_price_analysis.py
```

## Key Findings

See Complete_Analysis.md for detailed findings on:
- Price patterns across time of day, day of week, and seasons
- Location-specific price variations
- Optimization opportunities for flexible loads
